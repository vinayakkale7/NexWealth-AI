# NexWealth-AI Backend

This is the backend for NexWealth-AI. It will be built using FastAPI and PostgreSQL in future phases.

## Structure

- `app/api`: FastAPI route handlers
- `app/services`: Business logic
- `app/models`: Database models
- `app/schemas`: Pydantic schemas for request/response validation
- `app/database`: Database connection and session management
- `app/middleware`: Custom middleware
- `app/utils`: Utility functions
- `app/config`: Configuration settings
