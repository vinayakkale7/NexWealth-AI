# Setup Guide

## Prerequisites
- Node.js (v16+)
- Python (v3.10+) - Required for future backend development
- PostgreSQL - Required for future database integration

## Frontend Setup
1. Clone the repository.
2. Open terminal and navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```
3. Install frontend dependencies:
   ```bash
   npm install
   ```
4. Start the frontend development server:
   ```bash
   npm run dev
   ```
